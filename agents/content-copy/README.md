# Agent: content-copy

## Propósito
Este agente existe para producir copy comercial multicanal orientado a intención y claridad.

## Skills que consume
- `copywriting`
- `copy-editing`
- `cold-email`
- `email-sequence`
- `social-content`

## Casos de uso
- Diagnóstico inicial y plan accionable.
- Ejecución guiada por checklists de skills.
- Entrega de activos listos para revisión humana.

## Entradas mínimas
- `shared/product-marketing-context.md` completo.
- Objetivo de negocio medible (KPI + horizonte temporal).
- Restricciones (legal, marca, presupuesto, canal).

## Salidas esperadas
- Plan priorizado por impacto/esfuerzo.
- Artefactos reproducibles (briefs, checklists, tablas).
- Próximos pasos con dueños y criterios de aceptación.
