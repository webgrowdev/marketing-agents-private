# Integración con OpenClaw

Este workspace se sincroniza como `workspaces/motion-pro` y consume solo las skills de motion definidas para esta célula especializada.

## Skills permitidas
- `motion-designer`
- `svg-animation-engineer`
- `apple-ui-skills`
- `elevated-design`
- `video-motion-graphics`
- `dramatic-2000ms-plus`

## Reglas de ejecución
- Cargar primero `shared/product-marketing-context.md`.
- Priorizar coherencia de marca y legibilidad sobre efecto por efecto.
- Documentar constraints técnicos (FPS, peso, compatibilidad).
- Si falta skill curada, mantener placeholder explícito y escalar a importación offline.
